React use cases

In my opinion, the main benefits of React can be summarized in these four categories of features:

  • Component composition and reuse
  • DOM event handling
  • State management within your website
  • Reactive DOM content and attributes

Let's look at an example of each of these four features.





Component composition and reuse

All website pages are composed of HTML tags, with attributes and text content. Without a JavaScript framework like React, these tags are hardcoded into the webpage.

React lets users define "components", made of HTML tags. This approach lets pages be composed of well-defined components, rather than masses of HTML tags. Also, this approach lets users reuse components, maintaining some of that component's style and content, while replacing some data as needed.
