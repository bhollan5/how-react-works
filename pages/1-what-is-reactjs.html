What is ReactJS?

React is an open-source JavaScript framework, created by Facebook in 2013. It's used for building website UIs.


There are lots of options for how to design a website. The React "framework" gives you some powerful tools to design and manage bigger projects.


React lets users build "reactive" web pages, where changes to a single peice of data can propagate to multiple areas.









React Components

The fundamental building block of React's functionality is the React component.

All website pages are composed of HTML tags, with attributes and text content. Without a JavaScript framework like React, these tags are hardcoded into the webpage.

For example, here is the code for a simple button HTML tag, set up to call a Javascript function:


  <button onclick="hello_world()"\>
    Hello world!
  </button>
  <script>
    function hello_world() {
      alert("Hello world, from vanilla JS!");
    }
  </script>

That code produces this, with no ReactJS necessary:



I can create that component using React's syntax as well, like so:


  <!--  Adding the two necessary scripts to run React: -->
  <script src="https://unpkg.com/react@17/umd/react.development.js" crossorigin></script>  
  <script src="https://unpkg.com/react-dom@17/umd/react-dom.development.js" crossorigin></script>
  <!--  Here's where we'll insert out button component: -->
  <div id="button-demo"></div>
  <!--  And here's the Javascript we'll run to create our button component. -->
  <script>
      //  A function we want the button to call. 
      function hello_world() {
        alert("Hello world, from React!");
      }

      //  React's syntax to define our button component.
      function ButtonComponent() {
        return React.createElement(
          'button',                 //  The tag name
          { onClick: hello_world }, //  The element properties
          'Hello world!'            //  The element's inner content
        )
      }

      //  Rendering the button in the 'button-demo' div.
      ReactDOM.render(
        React.createElement(ButtonComponent),
        document.getElementById('button-demo')
      );
  </script>
