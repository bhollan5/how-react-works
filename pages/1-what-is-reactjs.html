What is ReactJS?

React is an open-source JavaScript framework, created by Facebook in 2013. It's used for building website UIs.


There are lots of options for how to design a website. The React "framework" gives you some powerful tools to help get you started.


React lets users build "reactive" web pages, where changes to a single peice of data can propagate to multiple areas.
