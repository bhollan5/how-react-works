Hello there!

Welcome to www.how-react.works. This website contains a guide to React, written by a person who is relatively new to React.


You may want to check out the official React documentation or the official React tutorial. I wrote this website's content by reading those tutorials, and re-writing the information I found most pertinant.


Unlike the official docs, I include working examples of React features on the website pages, rather than in separate codepen links.

This guide is also shorter than the official docs.


This guide includes the following sections:

1. What is React?

An intro to what React is, and how it can be used to build websites.

2. React use cases

Three examples of React code used to make website components.

3. JSX

Clean syntax for representing a React website with code.

4. How does React work?

Exploring the mechanics that let React work, like the virtual DOM.

5. Serving a React website

Server options for sending React website files to browsers.

6. Other React tools

A bit about some other tools you can add to a React website.
