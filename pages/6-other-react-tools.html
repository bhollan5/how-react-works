6. Other React tools

Aside from server tools, React can be used with several other tools.

On this page, I'll describe two popular tools that work well with React, and how they can be used:

  • Strongly typed JS with TypeScript
  • UI design tools with Material UI








Strongly typed JS with TypeScript

TypeScript is an extension of the JavaScript language that allows programmers to specify datatypes of variables. This means that TypeScript is considered a "strongly typed" language.

TypeScript's official docs have a page all about using TypeScript with various React setups.









UI design tools with Material UI

Material UI is a library of styled, functional React components, ready to be used in any React app.

The Material UI website provides straightforward instructions for adding Material UI to your website.









There are other tools and libraries that can be added to a ReactJS website, but for now, I'll leave it at those two.
